# GitFav
Esse projeto serve para adicionar a uma tabela seus usuários favoritos do GitHub, e para isso utilizei diversos novos conhecimentos que aprendi, dentre eles:
promises; 
conceitos da POO; 
api; 
fetch; 
try, catch e throw; 
async functions; 
imutabilidade;  
herança, dentre outros.
